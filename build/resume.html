Notable Projects

Mash.li http://mash.li

Mash.li is a content aggregate site built on top of the SoundCloud API. It scrapes the SoundCloud system for new tracks tagged as music mashups, and then queues them for regular release. Each day the site picks ten tracks at random from the queue and adds them to the top of the list, where visitors can listen to them.

Mash.li is written 100% in JavaScript on a NodeJS backend with a RequireJS, Backbone & Handlebars driven front-end. The codebase makes extensive use of Grunt tooling to manage front-end optimization for fast page loads. The entire codebase is made available as open source via GitHub.

Kalendae http://kalendaejs.com

Kalendae is a JavaScript date picker library written out of frustration with all other date pickers. The library is entirely self-contained and does not depend on any other DOM manipulation libraries such as jQuery or Prototype.js. It supports multiple types of date selection, including range selection, and is fully skinnable through CSS.
